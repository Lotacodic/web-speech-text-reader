# Web Speech Text Reader

A simple speech synthesis web application built with JavaScript and the Web Speech API that converts text into speech with selectable voices.

## Features

- Convert text into speech
- Select from available system voices
- Uses the Web Speech API
- Simple and clean user interface
- Works directly in the browser

## Built With

- HTML5
- CSS3
- JavaScript
- Web Speech API

## Project Structure

```
speech-synthesis-app
│
├── index.html
├── style.css
├── script.js
└── README.md
```

## How It Works

1. Enter any text in the input field.
2. Choose a voice from the dropdown list.
3. Click the **Speak** button.
4. The browser converts the text into speech using the selected voice.

## Getting Started

1. Clone the repository

```
git clone https://github.com/your-username/speech-synthesis-app.git
```

2. Open the project folder

```
cd speech-synthesis-app
```

3. Open **index.html** in your browser.

## Browser Support

The Web Speech API is supported in most modern browsers such as:

- Google Chrome
- Microsoft Edge
- Safari

Support may vary depending on the browser and operating system.
